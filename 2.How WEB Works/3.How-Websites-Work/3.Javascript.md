# JavaScript (JS)

## What is JavaScript?

JavaScript is a **programming language used to make websites interactive**.

---

## Role in Web Development

- **HTML** → structure (content)
- **CSS** → design (style)
- **JavaScript** → functionality (logic + interaction)

👉 Without JavaScript, websites are **static (non-interactive)**

---

## What JavaScript Can Do

- Respond to user actions (clicks, hover, typing)
- Change content dynamically
- Update page without reloading
- Create animations and effects

---

## How JavaScript is Added

### 1. Inside `<script>` tag
```html
<script>
  // JavaScript code here
</script>
```

---

### 2. External file (recommended)
```html
<script src="/location/of/script.js"></script>
```

---

## Example: Changing Page Content

```javascript
document.getElementById("demo").innerHTML = "Hack the Planet";
```

### What it does:
- Finds element with id `"demo"`
- Changes its text content

---

## Events in JavaScript

Events = actions performed by user

Examples:
- click
- hover
- keypress

---

## Example: Click Event

```html
<button onclick='document.getElementById("demo").innerHTML = "Button Clicked";'>
  Click Me!
</button>
```

---

## What Happens Here?

- User clicks button
- `onclick` event triggers JavaScript
- Text inside element changes

---

## Alternative Way

Events can also be written inside `<script>` instead of HTML tags.

---

## Key Takeaway

JavaScript makes websites **dynamic and interactive**, allowing real-time changes based on user actions.
