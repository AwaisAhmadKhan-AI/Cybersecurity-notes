# HTML (HyperText Markup Language)

## What is HTML?

HTML is the **main language used to build websites**.  
It defines the **structure and content** of a web page.

---

## Technologies Used in Websites

- **HTML** → structure (skeleton)
- **CSS** → design and styling
- **JavaScript** → interactivity and dynamic features

---

## Basic Idea

HTML is made of **elements (tags)**.  
These tags tell the browser how to display content.

---

## Basic HTML Structure

Every website follows a similar structure:

```html
<!DOCTYPE html>
<html>

<head>
    <title>Page Title</title>
</head>

<body>
    <h1>Main Heading</h1>
    <p>This is a paragraph.</p>
</body>

</html>
```

---

## Explanation of Elements

### `<!DOCTYPE html>`
- Declares the document as **HTML5**
- Ensures browsers interpret it correctly

---

### `<html>`
- Root element of the page
- Everything is inside this tag

---

### `<head>`
- Contains **metadata**
- Example: title, settings, links

---

### `<body>`
- Contains **visible content**
- Everything shown on the webpage

---

### `<h1>`
- Large heading (most important text)

---

### `<p>`
- Paragraph text

---

## HTML Tags & Attributes

### Example:
```html
<p class="bold-text">Hello</p>
```

### Common Attributes:

#### `class`
- Used for styling multiple elements
- Shared between elements

#### `id`
- Unique identifier for one element only
```html
<p id="example">Text</p>
```

#### `src`
- Used in images to define file path
```html
<img src="img/cat.jpg">
```

---

## Key Difference

| Attribute | Purpose |
|----------|--------|
| class | Used by multiple elements |
| id | Unique to one element |

---

## How to View HTML of a Website

1. Right-click on a webpage
2. Click **View Page Source**
3. See the HTML code

---

## Key Takeaway

HTML is the **foundation of every website**, used to define structure and content, while CSS and JavaScript add style and functionality.
