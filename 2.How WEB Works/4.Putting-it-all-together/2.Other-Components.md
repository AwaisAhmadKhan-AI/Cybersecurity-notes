# Other Components of the Web

Modern websites use extra systems to improve **performance, reliability, and security**.

---

## 1. Load Balancers

### What they do:
A **load balancer** distributes incoming traffic across multiple servers.

---

### Why they are needed:
- Prevents a single server from getting overloaded
- Ensures high availability (website stays online)

---

### How it works:
1. User sends request
2. Load balancer receives it
3. It forwards request to one of many servers

---

### Common algorithms:
- **Round Robin** → sends requests one by one to each server
- **Weighted** → sends more traffic to stronger/less busy servers

---

### Health Checks:
- Load balancer regularly checks servers
- If a server fails → it stops sending traffic to it

---

## 2. CDN (Content Delivery Network)

### What it does:
A **CDN stores copies of website files across multiple global servers**.

---

### What it stores:
- Images
- Videos
- CSS files
- JavaScript files

---

### Why it is used:
- Faster loading times
- Reduces distance between user and server
- Reduces load on main server

---

### How it works:
1. User requests a file
2. CDN finds nearest server
3. File is delivered from closest location

---

## 3. Databases

### What they do:
Databases store and manage website data.

---

### Examples:
- MySQL
- MSSQL
- MongoDB
- PostgreSQL

---

### What they store:
- User accounts
- Messages
- Website content
- Logs and records

---

### Key idea:
Web servers **read and write data** to databases when needed.

---

## 4. WAF (Web Application Firewall)

### What it does:
A **WAF protects websites from attacks and malicious traffic**.

---

### How it protects:
- Filters incoming requests
- Blocks suspicious traffic
- Detects bots and attacks
- Uses rate limiting (limits requests per second)

---

### Example protections:
- Stops DDoS-like traffic spikes
- Blocks SQL injection attempts
- Blocks suspicious bots

---

## Key Takeaway

Modern websites are not just servers — they are supported by:
- Load Balancers → distribute traffic
- CDNs → speed up content delivery
- Databases → store data
- WAFs → protect from attacks
