# Key Terminology

## Desktop

The main workspace where:

- Files
- Folders
- Shortcuts

are located.

---

## Taskbar

A control bar that provides access to:

- Applications
- System tools
- Settings
- Notifications

---

## Start Menu

The main navigation menu in Windows.

Used to access:

- Applications
- Settings
- Files
- Power options

---

## Search

A quick way to locate:

- Applications
- Files
- Folders
- Settings

using keywords.

---

## File Explorer

Windows' built-in file management tool.

Used to:

- Browse files
- Manage folders
- Copy and move data
- Search for content

---

## Windows Update

A built-in service that keeps:

- Windows OS
- Security features
- Microsoft applications

updated and secure.

---

## Microsoft Store

Windows' official application marketplace.

Used to:

- Install trusted applications
- Update Store apps
- Manage installed apps

---

## Windows Settings

The modern configuration center for Windows.

Used to manage:

- System settings
- Devices
- Personalization
- Updates
- Security

---

## Control Panel

The legacy administrative interface.

Provides access to:

- Advanced settings
- System configuration
- Administrative tools

---

## Task Manager

A monitoring tool that shows:

- Running applications
- Background processes
- CPU usage
- RAM usage
- System performance

---

## Windows Security

The central dashboard for managing Windows security features.

Includes:

- Antivirus protection
- Firewall settings
- Device security
- App protection

---

## Windows Defender Firewall

A built-in firewall that helps protect the system from unauthorized network traffic.

Functions:

- Monitors network connections
- Allows or blocks traffic
- Applies security rules
