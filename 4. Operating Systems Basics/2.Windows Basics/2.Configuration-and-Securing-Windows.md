# Configuration and Securing Windows

## Overview

Applications are programs that help users perform tasks such as:

- Browsing the internet
- Editing files
- Managing settings
- Running tools and games

Managing applications includes:

- Updating
- Installing
- Uninstalling

These tasks help keep a system secure, stable, and efficient.

---

# Updating Applications

## Why Updates Matter

Updates usually provide:

- Security patches
- Bug fixes
- Performance improvements
- New features

---

# Windows Update

## What is Windows Update?

A built-in Windows service that updates:

- Windows OS
- Security features
- Some Microsoft applications

### Benefits

- Improves security
- Fixes bugs
- Enhances stability

---

# Updating Third-Party Applications

Applications installed from other vendors may update differently.

### Common Methods

- Automatic updates
- Update prompts on launch
- Manual update checks
- Downloading a newer installer

## What is Task Manager?

A built-in monitoring tool used to view system activity in real time.

---

## Main Tabs

### Processes

Shows:

- Running applications
- Background processes
- Resource usage

---

### Performance

Displays:

- CPU usage
- RAM usage
- Disk activity
- Network activity

---

### Users

Shows:

- Logged-in users
- Resource consumption

---

### Details

Technical process information including:

- Process names
- Process IDs (PIDs)

---

### Services

Displays:

- Running services
- Stopped services

# Windows Security

## What is Windows Security?

A central dashboard used to manage system protection.

---

## Main Security Areas

### Virus & Threat Protection

Protects against:

- Viruses
- Malware
- Ransomware

Features:
- Real-time protection
- Manual scans
- Custom scans

---

### Firewall & Network Protection

Controls:

- Incoming traffic
- Outgoing traffic
- Network access

---

### App & Browser Control

Protects users from:

- Malicious websites
- Unsafe downloads
- Dangerous applications

---

### Device Security

Provides:

- Hardware-based protection
- Secure boot support
- Device integrity checks

---

# Malware Scanning

## Custom Scan

A custom scan allows scanning a specific folder or file.

### Steps

1. Open Windows Security
2. Select **Virus & Threat Protection**
3. Select **Scan Options**
4. Choose **Custom Scan**
5. Select target folder
6. Start scan

---

## Purpose

Used to:

- Detect malicious files
- Verify folder safety
- Investigate suspicious content

---

# Windows Defender Firewall

## What is a Firewall?

A firewall monitors and controls network traffic.

It decides:

- What traffic is allowed
- What traffic is blocked

---

## Firewall Network Profiles

### Domain Profile

Used when connected to an organization's network.

---

### Private Profile

Used for trusted networks such as:

- Home networks
- Lab environments

---

### Public Profile

Used for untrusted networks such as:

- Public Wi-Fi
- Airports
- Cafés

---

# Firewall Rules

Firewall rules determine how traffic is handled.

---

## Inbound Rules

Control traffic entering your computer.

Examples:
- Remote Desktop
- File Sharing

---

## Outbound Rules

Control traffic leaving your computer.

Examples:
- Web browsing
- Application updates

---

## Connection Rules

Control secure communications between systems.

---

# Advanced Firewall Features

Allows administrators to:

- View firewall rules
- Create new rules
- Modify existing rules
- Filter traffic
- Monitor connections

---

# Key Takeaway

Keeping Windows secure involves regularly updating software, managing installed applications, monitoring system activity with Task Manager, using Windows Security to detect threats, and controlling network traffic with Windows Defender Firewall.
