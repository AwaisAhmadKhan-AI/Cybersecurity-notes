# Operating Systems (Introduction)

## What is an Operating System (OS)?

An **Operating System (OS)** is software that acts as a bridge between the user, applications, and computer hardware.

Without an OS, the computer's hardware would be difficult to use directly.

---

## Why is an OS Important?

The operating system makes everyday tasks possible:

- Opening applications
- Playing music and videos
- Managing files and folders
- Connecting to the internet
- Using hardware devices

👉 It is the software that makes the computer usable.

---

## Main Role of an Operating System

The OS manages and coordinates all computer resources so that programs can run smoothly.

---

## Core Responsibilities

### Hardware Management
Controls and manages:
- CPU
- RAM
- Storage devices
- Network devices

---

### Process Management
- Starts programs
- Runs programs
- Stops programs
- Allocates resources to applications

---

### File Management
- Creates files and folders
- Organizes data
- Controls file access

---

### User Interface
Provides a way for users to interact with the computer:

- Graphical User Interface (GUI)
- Command Line Interface (CLI)

---

### Security
- User accounts
- Permissions
- Access control

---

## Common Operating Systems

### Windows
- Most popular desktop OS
- User-friendly
- Widely used in homes and businesses

---

### Linux
- Open-source
- Popular in servers and cybersecurity
- Highly customizable

---

### macOS
- Developed by Apple
- Used on Mac computers
- Known for stability and integration

---

### Android
- Mobile operating system by Google

---

### iOS
- Mobile operating system by Apple

---

## Scenario

If you receive an old computer, one of the first things you should identify is:

- Which operating system it runs
- Hardware specifications
- Current configuration

This information helps determine whether to:
- Upgrade it
- Reinstall the OS
- Sell it
- Use it for projects or learning

---

## Key Takeaway

An Operating System (OS) is the software layer that manages hardware, runs applications, organizes files, and provides a user-friendly way to interact with a computer.
