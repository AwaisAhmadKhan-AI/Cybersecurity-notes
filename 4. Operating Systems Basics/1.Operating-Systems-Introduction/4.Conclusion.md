## Key Terminology

### Operating System (OS)

The core software that:

- Manages hardware
- Runs applications
- Controls system resources
- Provides a way for users to interact with the computer

---

### Kernel Space

The most privileged part of the operating system.

Responsibilities:
- Direct hardware access
- CPU management
- Memory management
- Device control

👉 Home of the **Kernel**

---

### User Space

The area where normal applications run.

Characteristics:
- Limited permissions
- Cannot directly access hardware
- Uses system calls to communicate with the kernel

---

### Graphical User Interface (GUI)

The visual way of interacting with an operating system.

Examples:
- Windows
- Icons
- Menus
- Buttons

Used through:
- Clicking
- Tapping

---

### Command-Line Interface (CLI)

A text-based interface used to control the operating system through commands.

Advantages:
- Fast
- Precise
- Powerful

Widely used in:
- Linux
- System Administration
- Cybersecurity
