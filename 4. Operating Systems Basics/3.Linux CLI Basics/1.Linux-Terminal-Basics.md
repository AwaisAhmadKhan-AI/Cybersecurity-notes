# Linux Terminal Basics & File Navigation

## Overview

The Linux terminal is a text-based interface used to interact with the operating system by typing commands.

It is widely used in cybersecurity because it is:

- Fast
- Powerful
- Precise
- Required by many security tools

---

# What is a Terminal?

A **Terminal** allows users to communicate directly with Linux using commands instead of clicking icons.

### Why Cybersecurity Professionals Use It

- Faster than GUI navigation
- Greater control over the system
- Supports powerful security tools
- Useful for automation

---

# Basic Navigation Commands

## 1. pwd (Print Working Directory)

### Purpose

Shows your current location in the filesystem.

### Command

```bash
pwd
```

### Example Output

```bash
/home/ubuntu
```

### Remember

👉 "Where am I?"

---

## 2. ls (List Directory Contents)

### Purpose

Displays files and folders in the current directory.

### Command

```bash
ls
```

### Example

```bash
Desktop
Documents
Downloads
Pictures
Videos
```

---

## 3. ls -l (Long Listing)

### Purpose

Shows detailed information about files and directories.

### Command

```bash
ls -l
```

### Displays

- Permissions
- Owner
- File size
- Date modified
- File name

---

## 4. ls -al (Show Hidden Files)

### Purpose

Displays all files, including hidden files.

### Command

```bash
ls -al
```

### Hidden Files

Files beginning with a dot (`.`)

Examples:

```bash
.bashrc
.profile
.Xauthority
```

### Note

Hidden files are usually configuration files.

---

# Moving Through Directories

## 5. cd (Change Directory)

### Purpose

Move into another folder.

### Command

```bash
cd Documents
```

### Example

```bash
cd Documents
```

Moves into:

```bash
/home/ubuntu/Documents
```

---

## 6. cd ..

### Purpose

Move back one directory level.

### Command

```bash
cd ..
```

### Example

```bash
/home/ubuntu/Documents
```

becomes

```bash
/home/ubuntu
```

---

# Finding Files

## 7. find

### Purpose

Search for files and folders.

### Syntax

```bash
find <location> -name <filename>
```

### Example

```bash
find ~ -name mission_brief.txt
```

### Explanation

| Part | Meaning |
|---------|---------|
| find | Search utility |
| ~ | Home directory |
| -name | Search by filename |
| mission_brief.txt | Target file |

---

## Result

Linux returns the complete path of the file.

Example:

```bash
/home/ubuntu/projects/mission_brief.txt
```

---

# Reading Files

## 8. cat

### Purpose

Display the contents of a file.

### Command

```bash
cat mission_brief.txt
```

### Example

```bash
cat notes.txt
```

Displays everything inside the file.

# Key Takeaway

Linux navigation starts with a few essential commands: **pwd**, **ls**, **cd**, **find**, and **cat**. Mastering these commands allows you to move through the filesystem, locate files, and read information directly from the terminal.
