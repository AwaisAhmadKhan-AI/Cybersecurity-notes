# Iterations (While Loops)

## Overview

Our previous version of the number-guessing game could evaluate the user's guess and provide feedback. However, the game ended after a single guess, even if the answer was incorrect.

To make the game more interactive, we need a way to repeatedly ask the user for guesses until they find the correct number.

This is where **iterations** and **while loops** become useful.

---

## What is an Iteration?

An **iteration** is the repetition of a set of instructions.

In programming, loops allow us to execute the same block of code multiple times without rewriting it.

### Example

Instead of writing:

```javascript
console.log("Try again");
console.log("Try again");
console.log("Try again");
```

We can use a loop to repeat the instruction automatically.

---

## The While Loop

A **while loop** continues executing as long as a specified condition remains true.

### Syntax

```javascript
while (condition) {
    // code to repeat
}
```

### How It Works

1. Check the condition.
2. If the condition is true, execute the loop body.
3. Return to the condition.
4. Repeat until the condition becomes false.

---

## Using a While Loop in the Guessing Game

The game should continue until the user guesses the secret number.

We can express this condition as:

```javascript
while (guess !== secret) {
    // repeat until guess equals secret
}
```

### Understanding `!==`

The operator `!==` means **not equal to**.

Examples:

```javascript
5 !== 10   // true
```

```javascript
8 !== 8    // false
```

The loop continues running while the guess is different from the secret number.

---

## What Should Be Repeated?

Each time the user guesses incorrectly, we need to:

1. Ask the user for a guess.
2. Convert the input into a number.
3. Increase the try counter.
4. Compare the guess with the secret number.
5. Display feedback.
6. Repeat if the guess is still incorrect.

These instructions belong inside the loop.

---

## While Loop Implementation

```javascript
// Repeat until the user guesses the secret number.
while (guess !== secret) {
    const text = await rl.question("Take a guess: ");
    guess = parseInt(text, 10);

    tries = tries + 1;

    if (guess < 1 || guess > 20) {
        console.log("That number is out of range. Try again.");
    } else if (guess < secret) {
        console.log("Too low, try again.");
    } else if (guess > secret) {
        console.log("Too high, try again.");
    } else {
        console.log("You got it in", tries, "tries!");
    }
}
```

## Complete Program (Version 3)

```javascript
import * as readline from "node:readline/promises";
import { stdin as input, stdout as output } from "node:process";

const rl = readline.createInterface({ input, output });

try {
    const secret =
        Math.floor(Math.random() * 20) + 1; // 1 <= secret <= 20

    let tries = 0;
    let guess = 0; // start with a value that cannot be the secret

    console.log("I'm thinking of a number between 1 and 20");

    // Repeat until the user guesses the secret number.
    while (guess !== secret) {
        const text = await rl.question("Take a guess: ");
        guess = parseInt(text, 10);

        tries = tries + 1;

        if (guess < 1 || guess > 20) {
            console.log("That number is out of range. Try again.");
        } else if (guess < secret) {
            console.log("Too low, try again.");
        } else if (guess > secret) {
            console.log("Too high, try again.");
        } else {
            console.log("You got it in", tries, "tries!");
        }
    }

} finally {
    rl.close();
}
```

## Why Initialize `guess` to 0?

```javascript
let guess = 0;
```

The secret number is always between:

```javascript
1 <= secret <= 20
```

Since `0` can never be the secret number, the loop condition starts as:

```javascript
0 !== secret
```

which is always true.

This guarantees that the loop runs at least once.

---

## Key Concepts

* An **iteration** is one repetition of a loop.
* A **while loop** repeats code while a condition is true.
* `!==` means "not equal to".
* The loop stops when the condition becomes false.
* Loops help avoid repeating code manually.
* User input, validation, and feedback can all be repeated inside a loop.

---

## Summary

```javascript
while (guess !== secret) {
    // repeated code
}
```

### What Happens?

1. The user enters a guess.
2. The guess is converted into a number.
3. The number of tries increases.
4. The program compares the guess with the secret number.
5. Feedback is displayed.
6. The process repeats until the user guesses correctly.

The introduction of the `while` loop transforms the game from a one-attempt program into a fully interactive guessing game.
