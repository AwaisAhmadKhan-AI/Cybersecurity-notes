# Conditional Statements

## Overview

So far, our number-guessing game accepts a user's guess, but it does not provide any feedback about whether the guess is correct or incorrect.

To make the game interactive, we use **conditional statements** (`if`, `else if`, and `else`) to compare the user's guess with the secret number and provide appropriate feedback.

---

## Logic of the Program

The program follows these rules:

1. If the number is outside the valid range (1–20), inform the user that the number is out of range.
2. If the guess is less than the secret number, tell the user the guess is too low.
3. If the guess is greater than the secret number, tell the user the guess is too high.
4. If none of the above conditions are true, the guess must be equal to the secret number.

---

## Conditional Statements

```javascript
// Give a hint using if / else if / else.
if (guess < 1 || guess > 20) {
    console.log("That number is out of range. Try again.");
} else if (guess < secret) {
    console.log("Too low, try again.");
} else if (guess > secret) {
    console.log("Too high, try again.");
} else {
    console.log("You got it in", tries, "tries!");
}
```

---

## Understanding the Conditions

### 1. Checking the Valid Range

```javascript
if (guess < 1 || guess > 20)
```

The `||` operator means **OR**.

This condition becomes true when:

* The guess is less than `1`
* OR the guess is greater than `20`

### Example

```javascript
guess = 25;
```

Output:

```text
That number is out of range. Try again.
```

---

### 2. Checking if the Guess is Too Low

```javascript
else if (guess < secret)
```

This condition only runs if the previous condition was false.

If the user's guess is smaller than the secret number, the program displays:

```text
Too low, try again.
```

### Example

```javascript
secret = 15;
guess = 10;
```

Output:

```text
Too low, try again.
```

---

### 3. Checking if the Guess is Too High

```javascript
else if (guess > secret)
```

This condition executes only if all previous conditions are false.

If the guess is greater than the secret number, the program displays:

```text
Too high, try again.
```

### Example

```javascript
secret = 15;
guess = 18;
```

Output:

```text
Too high, try again.
```

---

### 4. Checking if the Guess is Correct

```javascript
else
```

The `else` block executes when all previous conditions are false.

If the guess is:

* Not less than 1
* Not greater than 20
* Not less than the secret
* Not greater than the secret

Then the guess must be equal to the secret number.

### Example

```javascript
secret = 15;
guess = 15;
```

Output:

```text
You got it in 1 tries!
```

---

## Why Use `else if`?

The conditions are **mutually exclusive**, meaning only one condition can be true at a time.

For example:

```javascript
guess = 25;
```

Since the guess is out of range, there is no need to check whether it is too high or too low.

Using `else if` improves efficiency because JavaScript stops checking conditions once a matching condition is found.

## Complete Program (Version 2)

```javascript
import * as readline from "node:readline/promises";
import { stdin as input, stdout as output } from "node:process";

const rl = readline.createInterface({ input, output });

try {
    const secret =
        Math.floor(Math.random() * 20) + 1; // 1 <= secret <= 20

    let tries = 0;
    let guess = 0;

    console.log("I'm thinking of a number between 1 and 20");

    const text = await rl.question("Take a guess: ");
    guess = parseInt(text, 10);

    tries = tries + 1;

    if (guess < 1 || guess > 20) {
        console.log("That number is out of range. Try again.");
    } else if (guess < secret) {
        console.log("Too low, try again.");
    } else if (guess > secret) {
        console.log("Too high, try again.");
    } else {
        console.log("You got it in", tries, "tries!");
    }

} finally {
    rl.close();
}
```

## Key Concepts

* `if` checks a condition.
* `else if` checks another condition if the previous one was false.
* `else` runs when all previous conditions are false.
* `||` means logical OR.
* Conditions allow programs to make decisions.
* Only one branch of an `if / else if / else` chain executes.
