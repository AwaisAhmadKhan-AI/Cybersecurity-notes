# Prompting the User for Input

## Overview

In the previous step, we:

* Created variables and constants.
* Generated a random secret number.
* Prepared the game logic for a number-guessing game.

Now, we need to allow the user to enter a guess and convert that input into a number that JavaScript can work with.

---

## Reading User Input

```javascript
const text = await rl.question("Take a guess: ");
guess = parseInt(text, 10);
```

### Line 1: Getting Input

```javascript
const text = await rl.question("Take a guess: ");
```

* `rl.question()` displays a prompt to the user.
* The program pauses and waits for the user to type something.
* Whatever the user enters is returned as a **string (text)**.
* The `await` keyword waits until the user provides input.

**Example:**

```text
Take a guess: 15
```

After the input:

```javascript
text = "15";
```

---

### Line 2: Converting Text to a Number

```javascript
guess = parseInt(text, 10);
```

The `parseInt()` function converts text into an integer.

#### Syntax

```javascript
parseInt(value, radix);
```

| Parameter | Description                     |
| --------- | ------------------------------- |
| `value`   | The string to convert           |
| `radix`   | The number system (base) to use |

In this case:

```javascript
parseInt(text, 10);
```

* `text` contains the user's input.
* `10` specifies that the number is in **decimal (base 10)**.

---

## Why Use Base 10?

Computers can interpret numbers in different bases:

| Base | Number System |
| ---- | ------------- |
| 2    | Binary        |
| 8    | Octal         |
| 10   | Decimal       |
| 16   | Hexadecimal   |

Since users normally enter decimal numbers, we use:

```javascript
parseInt(text, 10);
```

to ensure correct conversion.

---

## Example

### User Input

```text
Take a guess: 7
```

### Internal Processing

```javascript
text = "7";
guess = parseInt(text, 10);
```

### Result

```javascript
guess = 7;
```

The value is now a number and can be compared with the secret number.

## Updating the Try Counter

```javascript
tries = tries + 1;
```

This increases the number of attempts made by the user.

### Example

Before:

```javascript
tries = 0;
```

After:

```javascript
tries = 1;
```

# Complete Program (Version 1)

```javascript
import * as readline from "node:readline/promises";
import { stdin as input, stdout as output } from "node:process";

const rl = readline.createInterface({ input, output });

try {
    const secret = Math.floor(Math.random() * (20)) + 1; // 1 <= secret <= 20
    let tries = 0;
    let guess = 0; // start with a value that cannot be the secret

    console.log("I'm thinking of a number between 1 and 20");

    const text = await rl.question("Take a guess: ");
    guess = parseInt(text, 10);

    tries = tries + 1;

} finally {
    rl.close();
}
```

---

# Key Concepts Learned

* `rl.question()` is used to collect input from the user.
* User input is always returned as a string.
* `parseInt()` converts a string into an integer.
* Using base `10` ensures decimal number conversion.
* `await` pauses execution until input is received.
* The `tries` variable keeps track of the number of guesses.
