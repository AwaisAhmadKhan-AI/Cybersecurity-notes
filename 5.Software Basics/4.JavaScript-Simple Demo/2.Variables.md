# Variables

## Setting the Stage

JavaScript files can be executed in different ways:

- Inside a web browser
- Using Node.js

For this room, Node.js is used because it is the easiest way to run JavaScript programs from the command line.

### Running JavaScript with Node.js

A JavaScript file can be executed using:

```bash
node demo.js
```

Where `demo.js` is the JavaScript filename.

---

## Running JavaScript in a Browser

JavaScript can also run directly in a web browser.

### Steps

1. Open your browser
2. Press `F12`
3. Open the **Developer Tools**
4. Go to the **Console**

You can then type JavaScript commands directly.

Example:

```javascript
console.log("Hello World");
```

---

# Creating Variables

Variables are memory locations used to store values that can change during program execution.

In JavaScript, variables are declared using the `let` keyword.

Example:

```javascript
let tries = 0;
let guess = 0;
```

---

## Explanation

### `tries`

```javascript
let tries = 0;
```

- Stores the number of attempts made by the player.
- Starts at `0` because the player has not guessed yet.

### `guess`

```javascript
let guess = 0;
```

- Stores the player's guessed number.
- Initialized to `0` because the secret number is between `1` and `20`.

---

# Creating Constants

Variables can change, but constants remain the same during the program.

JavaScript uses the `const` keyword for constants.

Example:

```javascript
const secret = 12;
```

---

# Random Secret Number

To generate a random number between `1` and `20`:

```javascript
const secret = Math.floor(Math.random() * (20)) + 1;
```

---

## Breakdown of the Expression

### Step 1 — Generate Random Decimal

```javascript
Math.random()
```

Produces a random decimal between:

```text
0 <= number < 1
```

Example:

```javascript
0.372
```

---

### Step 2 — Multiply by 20

```javascript
Math.random() * 20
```

Expands the range:

```text
0 <= number < 20
```

Example:

```javascript
7.44
```

---

### Step 3 — Remove Decimal Part

```javascript
Math.floor()
```

Rounds the number down.

Example:

```javascript
7.44 → 7
```

---

### Step 4 — Shift Range

```javascript
+ 1
```

Changes the range from:

```text
0–19
```

to:

```text
1–20
```

---

## Final Result

```javascript
const secret = Math.floor(Math.random() * (20)) + 1;
```

Possible values:

```text
1, 2, 3, ..., 20
```

# Displaying Output

JavaScript uses `console.log()` to display output on the screen.

Example:

```javascript
console.log("I'm thinking of a number between 1 and 20");
```

This prints the message to the terminal or browser console.

| Concept | Description |
