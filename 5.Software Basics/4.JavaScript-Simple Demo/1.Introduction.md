# Introduction to JavaScript

## Overview

JavaScript is one of the most popular programming languages and is widely used in web development.

Originally, JavaScript was designed to run inside web browsers on the client side. Today, with technologies like Node.js, JavaScript can also run on servers.

This means JavaScript is now used for:

- Interactive websites
- Web applications
- Backend/server development
- APIs
- Automation scripts

# Guess the Number Game

In this room, we create a simple game called **Guess the Number**.

## Game Logic

1. The computer secretly selects a number between 1 and 20.
2. The player keeps guessing the number.
3. The program gives hints:
   - "Too high"
   - "Too low"
4. The game ends when the correct number is guessed.
