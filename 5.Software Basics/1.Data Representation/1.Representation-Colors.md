# Representing Colors

## Overview

Computers create colors by combining three primary colors:

- Red (R)
- Green (G)
- Blue (B)

This is called the **RGB Color Model**.

Think of RGB as three light bulbs whose brightness can be adjusted.

---

# The First 8 Colors

## Using Only ON and OFF

Suppose each color can only be:

- ON (1)
- OFF (0)

Since there are 3 colors:

```text
2 × 2 × 2 = 8 colors
```

Each color is represented using **3 bits**:

```text
RGB
```

---

# 8 Basic Colors

| Binary | Meaning | Color |
|----------|----------|---------|
| 000 | All OFF | Black |
| 001 | Blue ON | Blue |
| 010 | Green ON | Green |
| 100 | Red ON | Red |
| 011 | Green + Blue | Cyan |
| 101 | Red + Blue | Magenta |
| 110 | Red + Green | Yellow |
| 111 | All ON | White |

---

# What is a Bit?

A **bit** is the smallest unit of data.

It can only be:

```text
0 = OFF
1 = ON
```

Examples:

```text
0
1
```

---

# From 8 Colors to 16 Million Colors

Using only ON and OFF gives very few colors.

Modern computers use:

```text
256 levels
```

for each color.

### Red

```text
0 → 255
```

### Green

```text
0 → 255
```

### Blue

```text
0 → 255
```

---

# Total Colors

```text
256 × 256 × 256
=
16,777,216 colors
```

Over **16 million colors**.

---

# What is a Byte?

A **byte** consists of:

```text
8 bits
```

Example:

```text
10100011
```

Since 8 bits can represent:

```text
256 values
(0–255)
```

one color channel uses 1 byte.

---

# Color Size in Memory

A color uses:

```text
Red   = 1 Byte
Green = 1 Byte
Blue  = 1 Byte
```

Total:

```text
3 Bytes
=
24 Bits
```

This is called **24-bit Color**.

---

# Example RGB Color

Binary:

```text
10100011 11101010 00101010
```

This represents:

```text
Red   = 10100011
Green = 11101010
Blue  = 00101010
```

---

# Why Hexadecimal?

Writing long binary numbers is difficult.

Example:

```text
101000111110101000101010
```

Hard to read ❌

So computers use **Hexadecimal (Hex)**.

---

# Example Conversion

Binary:

```text
10100011 11101010 00101010
```

Split into groups of 4 bits:

```text
1010 0011 1110 1010 0010 1010
```

Convert each group:

```text
1010 = A
0011 = 3
1110 = E
1010 = A
0010 = 2
1010 = A
```

Result:

```text
A3EA2A
```

# Color Representation Summary

A computer color is usually:

```text
24 bits
=
3 bytes
```

Where:

```text
1 byte = Red
1 byte = Green
1 byte = Blue
```

Each byte:

```text
0–255 intensity
```

# Key Takeaway

Computers create colors using **RGB (Red, Green, Blue)**.

- 1 Bit → 2 values (0 or 1)
- 1 Byte → 256 values (0–255)
- RGB uses 3 Bytes (24 Bits)
- Total possible colors:

```text
16,777,216
```

Hexadecimal makes color values shorter and easier to read:

```text
A3EA2A
```

instead of:

```text
101000111110101000101010
```
