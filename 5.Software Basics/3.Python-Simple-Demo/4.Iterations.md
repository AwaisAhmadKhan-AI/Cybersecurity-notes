# Iterations (Loops)

## Overview

Iterations, commonly called **loops**, allow a program to repeat the same block of code multiple times.

Instead of running a set of instructions once, a loop keeps executing them until a specific condition becomes false.

In the **Guess the Number** game, loops allow the player to keep guessing until the correct number is found.

---

# Why Do We Need Loops?

Without loops, the player only gets one chance to guess.

Example:

```text
Take a guess: 5
Too low, try again.
```

The program immediately ends.

This makes the game difficult and unrealistic.

Using a loop allows the player to continue guessing until they succeed.

# Iteration in the Guessing Game

The game should continue asking for guesses until:

```python
guess == secret
```

As long as the guess is incorrect:

```python
guess != secret
```

the program should continue running.

# The While Loop

Python uses the `while` statement for repetition.

## Syntax

```python
while condition:
    # code to repeat
```

The loop continues as long as the condition remains true.

---

# While Loop Used in the Game

```python
while guess != secret:
```

### Meaning

```text
Keep running while the guess is not equal to the secret number.
```

---

# Loop Structure

```python
while guess != secret:
    text = input("Take a guess: ")
    guess = int(text)

    tries = tries + 1

    if guess < 1 or guess > 20:
        print("That number is out of range. Try again.")
    elif guess < secret:
        print("Too low, try again.")
    elif guess > secret:
        print("Too high, try again.")
    else:
        print("You got it in", tries, "tries!")
```

---

# How the Loop Works

### Step 1

Check:

```python
guess != secret
```

---

### Step 2

If True:

```text
Run the code inside the loop
```

---

### Step 3

Ask for a new guess.

```python
input()
```

---

### Step 4

Compare the guess with the secret number.

---

### Step 5

Repeat the process.

---

### Step 6

When:

```python
guess == secret
```

the loop stops.

# Complete Program (Version 3)

```python
import random

# Guess the Number Game

secret = random.randint(1, 20)
tries = 0
guess = 0

print("I'm thinking of a number between 1 and 20")

while guess != secret:
    text = input("Take a guess: ")
    guess = int(text)

    tries = tries + 1

    if guess < 1 or guess > 20:
        print("That number is out of range. Try again.")
    elif guess < secret:
        print("Too low, try again.")
    elif guess > secret:
        print("Too high, try again.")
    else:
        print("You got it in", tries, "tries!")
```

# Summary

- Iteration means repeating a block of code.
- Loops allow programs to perform repetitive tasks automatically.
- Python uses the `while` loop for repeated execution.
- The operator `!=` means "not equal to".
- The game keeps running while:
