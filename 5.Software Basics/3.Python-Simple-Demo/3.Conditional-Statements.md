# Conditional Statements

## Overview

Conditional statements allow a program to make decisions based on conditions.

In the **Guess the Number** game, conditional statements are used to compare:

- The user's guess (`guess`)
- The secret number (`secret`)

Based on the comparison, the program provides feedback.

---

# Why Conditional Statements Are Needed

After receiving a guess, the program must determine:

1. Is the number outside the valid range (1–20)?
2. Is the guess too low?
3. Is the guess too high?
4. Is the guess correct?

Without conditional statements, the program cannot decide what message to display.

---

# Pseudocode

Before writing code, programmers often write pseudocode (human-readable logic).

```text
If guess is less than 1 or greater than 20:
    Print "Out of range"

Else if guess is less than secret:
    Print "Too low"

Else if guess is greater than secret:
    Print "Too high"

Else:
    Print "You got it"
```

---

# Python Conditional Statements

Python uses:

```python
if
elif
else
```

### Syntax

```python
if condition:
    # code

elif condition:
    # code

else:
    # code
```

# Understanding the Conditions

## 1. Out of Range Check

```python
if guess < 1 or guess > 20:
```

### Meaning

The guess is invalid if:

- Less than 1
- Greater than 20

### Example

```python
guess = 30
```

Output:

```text
That number is out of range. Try again.
```

---

# Logical Operator: OR

Python uses `or` when either condition can be true.

### Syntax

```python
condition1 or condition2
```

### Example

```python
guess < 1 or guess > 20
```

Meaning:

```text
Less than 1 OR greater than 20
```

---

# 2. Too Low Check

```python
elif guess < secret:
```

### Meaning

The guessed number is smaller than the secret number.

### Example

```python
secret = 10
guess = 5
```

Output:

```text
Too low, try again.
```

---

# 3. Too High Check

```python
elif guess > secret:
```

### Meaning

The guessed number is larger than the secret number.

### Example

```python
secret = 10
guess = 15
```

Output:

```text
Too high, try again.
```

---

# 4. Correct Guess

```python
else:
```

If all previous conditions are false, then:

```text
guess == secret
```

must be true.

### Example

```python
secret = 10
guess = 10
```

Output:

```text
You got it in 1 tries!
```

# Complete Program (Version 2)

```python
import random

secret = random.randint(1, 20)
tries = 0
guess = 0

print("I'm thinking of a number between 1 and 20")

text = input("Take a guess: ")
guess = int(text)

tries = tries + 1

if guess < 1 or guess > 20:
    print("That number is out of range. Try again.")
elif guess < secret:
    print("Too low, try again.")
elif guess > secret:
    print("Too high, try again.")
else:
    print("You got it in", tries, "tries!")
```
