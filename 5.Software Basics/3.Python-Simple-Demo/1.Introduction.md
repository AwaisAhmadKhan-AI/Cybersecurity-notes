# Introduction to Python

## What is Python?

Python is a **high-level, general-purpose programming language**.

### High-Level Language
- Hides most low-level implementation details.
- Easier for humans to read and write.
- Allows programmers to focus on solving problems rather than managing hardware.

### General-Purpose Language
Python can be used for many different tasks, including:

- Web Development
- Automation & Scripting
- Cybersecurity Tools
- Data Science
- Machine Learning
- Artificial Intelligence
- Software Development

# Project: Guess the Number Game

In this room, we will create a simple game called **Guess the Number**.

### Game Rules

1. The computer secretly chooses a number between **1 and 20**.
2. The player enters a guess.
3. The computer checks the guess.
4. If the guess is:
   - Too high → Computer says **"Too high"**
   - Too low → Computer says **"Too low"**
5. The game continues until the correct number is guessed.
6. The computer displays how many attempts were required.

### Example Output

```text
I'm thinking of a number between 1 and 20
Take a guess: 10
Too high, try again.

Take a guess: 5
Too low, try again.

Take a guess: 7
Too low, try again.

Take a guess: 8
You got it in 4 tries!
