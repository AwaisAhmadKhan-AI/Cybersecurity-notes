# Variables

## Overview

Variables are used to store data in a program. In this project, three variables are needed:

- `secret` → Stores the random number chosen by the computer.
- `guess` → Stores the user's guess.
- `tries` → Counts how many attempts the user has made.

---

# Generating a Random Number

Python provides the `random.randint()` function to generate a random integer within a specified range.

## Syntax

```python
random.randint(start, end)
```

### Example

```python
secret = random.randint(1, 20)
```

This generates a random number between **1 and 20** (inclusive).

---

# Initializing Variables

```python
secret = random.randint(1, 20)
tries = 0
guess = 0
```

### Explanation

| Variable | Purpose | Initial Value |
|----------|----------|----------|
| `secret` | Stores the random secret number | Random number (1-20) |
| `tries` | Counts attempts | 0 |
| `guess` | Stores user guess | 0 |

### Why is `guess` set to 0?

Because the valid range is **1–20**, using `0` ensures it cannot accidentally match the secret number.

---

# Displaying Information

Python uses the `print()` function to display output.

## Example

```python
print("I'm thinking of a number between 1 and 20")
```

### Output

```text
I'm thinking of a number between 1 and 20
```

---

# Taking User Input

Python uses the `input()` function to receive input from the user.

## Example

```python
text = input("Take a guess: ")
```

### Output

```text
Take a guess:
```

The user's input is stored as **text (string)**.

---

# Converting Text to a Number

Since `input()` always returns text, we must convert it to an integer using `int()`.

## Example

```python
guess = int(text)
```

### Example Conversion

```python
text = "8"
guess = int(text)
```

Result:

```python
guess = 8
```

---

# Counting Attempts

Each time the user enters a guess, the attempt counter increases.

## Example

```python
tries = tries + 1
```

### How It Works

Before:

```python
tries = 0
```

After one guess:

```python
tries = 1
```

After another guess:

```python
tries = 2
```

And so on.
