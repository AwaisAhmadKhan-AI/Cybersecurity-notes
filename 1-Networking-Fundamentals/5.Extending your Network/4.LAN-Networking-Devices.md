# LAN Networking Devices

## What is a Router?

A **router** is a networking device that connects **different networks** and forwards data between them.

---

## Main Role of a Router

- Connects multiple networks together
- Sends data between networks
- Chooses the best path for data (routing)

---

## What is Routing?

**Routing** is the process of finding the best path for data to travel between networks.

Routers decide paths based on:

- Shortest route
- Most reliable route
- Fastest connection type (e.g., fibre vs copper)

---

## OSI Layer

- Routers operate at **Layer 3 (Network Layer)**

---

## Router Features

- Can manage multiple network paths
- Supports configuration (via web interface or console)
- Can set rules like:
  - Port forwarding
  - Firewall rules

---

## Simple Idea

If devices are in different networks:
- Router acts as the “bridge” between them

---

# What is a Switch?

A **switch** is a device used to connect multiple devices within the same network.

Example devices:
- PCs
- Printers
- Servers

---

## Switch Capacity

- Can connect multiple devices using Ethernet cables
- Typically supports many ports (3 to 63+ devices)

---

## Layer 2 Switch

### How it works:
- Operates at **Layer 2 (Data Link Layer)**
- Uses **MAC addresses**
- Sends **frames** to correct device

👉 Think of it as: “send data to the correct device inside a network”

---

## Layer 3 Switch

### How it works:
- Works at **Layer 2 + Layer 3**
- Uses:
  - MAC addresses (for local delivery)
  - IP addresses (for routing)

👉 It behaves like a mix of:
- Switch + Router

---

## VLAN (Virtual Local Area Network)

### What is VLAN?

A **VLAN** is a way to logically split a single physical network into multiple virtual networks.

---

## Why VLAN is Used

- Improves security
- Separates departments
- Controls communication between groups

---

## Example

- Sales Department VLAN
- Accounting Department VLAN

Both:
- Can access the internet
- Cannot communicate with each other

👉 Even if they are connected to the same switch

---

## Key Difference: Switch vs Router

| Router | Switch |
|--------|--------|
| Connects different networks | Connects devices in same network |
| Uses IP addresses | Uses MAC addresses |
| Layer 3 device | Layer 2 / Layer 3 device |
| Performs routing | Forwards frames |

---

## Key Takeaway

- **Router = connects networks**
- **Switch = connects devices inside a network**
- **Layer 3 switches + VLANs add intelligence and segmentation inside networks**
