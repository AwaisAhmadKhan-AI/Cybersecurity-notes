# Intro to Port Forwarding

## What is Port Forwarding?

**Port forwarding** is a technique used to allow external devices (on the internet) to access services inside a private network.

Without it:
- Services stay inside the local network only
- External users cannot access them

---

## Why Port Forwarding is Needed

Inside a private network:
- Devices communicate using private IP addresses (e.g., `192.168.1.10`)
- These devices are not directly accessible from the internet

Port forwarding allows:
- External traffic to reach a specific device and service inside the network

---

## Simple Example

### Without Port Forwarding (Private Network / Intranet)

- Web server IP: `192.168.1.10`
- Service: HTTP on port 80
- Only local devices can access it

👉 This is called an **intranet** (internal network only)

---

### With Port Forwarding

- Router receives internet traffic
- Forwards specific port traffic to internal device

Example:
- Public IP: `82.62.51.70`
- Requests to port 80 are forwarded to `192.168.1.10`

Now:
- External users can access the web server via the internet

---

## How It Works (Simple Flow)

1. User sends request to public IP
2. Router receives request
3. Router checks port forwarding rules
4. Router forwards traffic to internal device
5. Device responds back through router

---

## Port Forwarding vs Firewall

### Port Forwarding
- Opens access to specific internal services
- Allows external traffic in

### Firewall
- Controls whether traffic is allowed or blocked
- Can block traffic even if port forwarding exists

👉 Port forwarding = opens door  
👉 Firewall = guards the door

---

## Where Port Forwarding is Configured

- Configured on the **router**
- Router decides where incoming traffic should go inside the network

---

## Key Takeaway

Port forwarding allows external users on the internet to access specific services inside a private network by routing traffic through the router to the correct internal device.
