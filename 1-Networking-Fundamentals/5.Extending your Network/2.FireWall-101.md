# Firewall 101

## What is a Firewall?

A **firewall** is a security system that controls network traffic by deciding what is allowed to enter or leave a network.

👉 Think of it as a **security guard at a border checkpoint**.

---

## Main Role

A firewall checks network traffic and decides whether to:

- Allow it (permit)
- Block it (deny)

---

## What Firewalls Check

Firewalls inspect data based on several factors:

### 1. Source of Traffic
- Where is the traffic coming from?
- Is it from a trusted network or blocked network?

---

### 2. Destination of Traffic
- Where is the traffic going?
- Should this destination be allowed?

---

### 3. Port Number
- Which service is being accessed?
- Example: Port 80 (HTTP), Port 443 (HTTPS)

---

### 4. Protocol Type
- TCP, UDP, or other protocols
- Firewall may allow or block based on protocol

---

## Packet Inspection

Firewalls perform **packet inspection** to analyze incoming and outgoing data.

This helps them decide:
- What to allow
- What to block

---

## Types of Firewalls

### 1. Stateful Firewall

- Tracks the **entire connection**
- Understands context of communication
- Makes decisions based on connection history

#### Key Features:
- Intelligent decision-making
- Blocks entire malicious connections
- More secure but uses more resources

#### Example:
- If a connection behaves suspiciously → entire session is blocked

---

### 2. Stateless Firewall

- Examines each **packet individually**
- Uses fixed rules only
- Does NOT track connection state

#### Key Features:
- Fast and lightweight
- Less resource usage
- Less intelligent (rule-based only)

#### Example:
- A bad packet may be blocked, but the rest of the connection is still allowed

---

## Stateful vs Stateless (Simple Comparison)

| Stateful | Stateless |
|----------|----------|
| Tracks full connection | Checks individual packets |
| More intelligent | Rule-based only |
| Uses more resources | Uses fewer resources |
| More secure | Less secure but faster |

---

## Where Firewalls Are Found

Firewalls can exist in different forms:

- Hardware firewalls (enterprise networks)
- Software firewalls (on devices)
- Home routers (built-in firewall systems)

---

## Key Takeaway

A firewall is a **security system that filters network traffic**, using rules like IP address, port, and protocol to decide what is safe or unsafe.
