# OSI Layer 3 - Network Layer

## Overview
The **Network Layer** is responsible for **routing data** and breaking/reassembling it for delivery across different networks.

---

## Main Role
- Finds the best path for data to travel
- Uses **IP addresses** for communication
- Helps move data between different networks

---

## Routing

**Routing** means choosing the best path for data to reach its destination.

---

## How the Best Path is Chosen

Routers decide the path based on:

- Shortest path (fewest devices)
- Most reliable path (less packet loss)
- Fastest connection (fiber is faster than copper)

---

## Routing Protocols (Basic Idea)

Some protocols used for routing:

- OSPF (Open Shortest Path First)
- RIP (Routing Information Protocol)

*(Just know they help routers find the best path.)*

---

## IP Address Usage

At this layer, communication is done using **IP addresses**, for example:

- `192.168.1.100`

---

## Layer 3 Devices

Devices that work at this layer are called:

- **Routers**

They use IP addresses to send data between networks.

---

## Key Takeaway

The **Network Layer (Layer 3)** is responsible for routing data using IP addresses and choosing the best path for data to travel between networks.````
