# OSI Layer 7 - Application Layer

## Overview
The **Application Layer** is the top layer of the OSI Model and is the layer users interact with directly.

It provides services and protocols that allow applications to access the network.

---

## Main Role
- Provides interface between user and network
- Enables applications to send and receive data
- Defines rules for network services used by software

---

## Where You See It

This layer includes everyday applications like:
- Web browsers
- Email clients
- File transfer tools (e.g., FileZilla)

These applications provide a **Graphical User Interface (GUI)** so users can easily interact with data.

---

## Important Protocols

### DNS (Domain Name System)
- Converts domain names into IP addresses
- Example:
  - `google.com → 142.250.x.x`

---

## Key Functions

- User interaction with network services
- Data request and response handling
- Application-level communication rules

---

## Key Takeaway

The **Application Layer (Layer 7)** is the top OSI layer where users interact with network services through applications like browsers, email clients, and DNS services.
