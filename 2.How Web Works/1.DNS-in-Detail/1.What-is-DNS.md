# What is DNS?

## Overview

**DNS (Domain Name System)** is a system that translates human-friendly website names into computer-readable IP addresses.

---

## Why DNS is Needed

Computers communicate using IP addresses like:

- `104.26.10.229`

But humans find it hard to remember numbers.

So instead of IPs, we use domain names like:

- `tryhackme.com`

👉 DNS connects the two.

---

## Simple Analogy

Think of DNS like a **phonebook**:

- Name = Website (e.g., google.com)
- Number = IP address (e.g., 142.250.x.x)

You search the name → DNS gives the number.

---

## How DNS Works (Simple Flow)

1. You type a website (e.g., `google.com`)
2. DNS receives the request
3. DNS finds the matching IP address
4. Your browser connects to that IP
5. Website loads

---

## Example

| Domain Name | IP Address |
|-------------|------------|
| tryhackme.com | 104.26.10.229 |

---

## Key Idea

- DNS removes the need to memorize IP addresses
- It makes browsing the internet simple and human-friendly

---

## Key Takeaway

**DNS translates domain names into IP addresses so humans can access websites without remembering complex numbers.**
