# Domain Hierarchy

## Overview

A domain name is structured in a hierarchy that helps organize how websites are named and accessed on the internet.

It is read from **right to left**.

---

## Domain Structure Example

Example:
```
admin.servers.tryhackme.com
```

Breakdown:
- `.com` → Top-Level Domain (TLD)
- `tryhackme` → Second-Level Domain (SLD)
- `admin` / `servers` → Subdomains

---

## 1. Top-Level Domain (TLD)

### What it is:
The **rightmost part** of a domain name.

Example:
- `tryhackme.com` → `.com` is the TLD

---

### Types of TLDs:

#### gTLD (Generic Top-Level Domain)
Used for general purpose:

- `.com` → commercial websites
- `.org` → organizations
- `.edu` → education
- `.gov` → government

Modern gTLDs also include:
- `.online`
- `.website`
- `.biz`
- `.club`

---

#### ccTLD (Country Code Top-Level Domain)
Used for countries:

- `.ca` → Canada
- `.uk` → United Kingdom
- `.pk` → Pakistan

---

## 2. Second-Level Domain (SLD)

### What it is:
The **main name** of the website (left of the TLD).

Example:
- `tryhackme.com`
  - `tryhackme` = SLD

---

### Rules:
- Max 63 characters
- Can use:
  - a–z
  - 0–9
  - hyphens (-)
- Cannot:
  - start or end with hyphens
  - use consecutive hyphens

---

## 3. Subdomain

### What it is:
A part added to the **left side of the SLD**.

Example:
- `admin.tryhackme.com`
  - `admin` = subdomain

---

### Key Points:
- Used to organize services (e.g., admin panels, servers)
- Can have multiple levels:

Example:
```
jupiter.servers.tryhackme.com
```

---

### Rules:
- Same rules as SLD
- Max 63 characters per subdomain
- Total domain length max 253 characters
- No limit on number of subdomains

---

## Simple Analogy

Think of a domain like an address:

- TLD = Country
- SLD = City
- Subdomain = Street / Building

---

## Key Takeaway

Domain hierarchy helps organize websites:
- **TLD → type or country**
- **SLD → main website name**
- **Subdomain → sections or services of a website**
