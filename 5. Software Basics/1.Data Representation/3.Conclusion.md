# Number Systems Covered

## Decimal System (Base-10)

- The numbering system used in everyday life.
- Uses digits:

```text
0 - 9
```

- Each position represents a power of 10.

Example:

```text
213 = 2×10² + 1×10¹ + 3×10⁰
```

---

## Binary System (Base-2)

- The language computers understand.
- Uses only:

```text
0 and 1
```

- Represents two states such as:
  - ON / OFF
  - TRUE / FALSE
  - High Voltage / Low Voltage

Example:

```text
1001₂ = 9₁₀
```

---

## Hexadecimal System (Base-16)

- A compact way to represent binary values.
- Uses:

```text
0-9 and A-F
```

Where:

```text
A = 10
B = 11
C = 12
D = 13
E = 14
F = 15
```

- Every **4 binary bits** equal **1 hexadecimal digit**.

Example:

```text
1010 = A
1111 = F
```

---

## Octal System (Base-8)

- Uses digits:

```text
0 - 7
```

- Every **3 binary bits** equal **1 octal digit**.

Example:

```text
111₂ = 7₈
```

- Less commonly used in modern computing.

---

# Data Representation Concepts

## Bit

**Bit** stands for **Binary Digit**.

A bit can only have two values:

```text
0 or 1
```

Examples:

```text
0 = OFF
1 = ON
```

---

## Byte

A **Byte** consists of:

```text
8 bits
```

A byte can represent:

```text
256 different values
```

Range:

```text
0 - 255
```

A byte is also called an:

```text
Octet
```

---

# Color Representation in Computers

Computers create colors using three primary colors:

- Red (R)
- Green (G)
- Blue (B)

This is known as:

```text
RGB Color Model
```

---

## RGB Structure

Each color receives:

```text
1 Byte (8 bits)
```

Therefore:

```text
Red   = 8 bits
Green = 8 bits
Blue  = 8 bits
```

Total:

```text
24 bits
```

or

```text
3 Bytes
```

---

## Number of Possible Colors

Each color channel can have:

```text
256 values
```

Calculation:

```text
256 × 256 × 256
```

Result:

```text
16,777,216 colors
```

More than:

```text
16 Million Colors
```

---

## Hex Color Format

RGB colors are commonly written in hexadecimal format:

Example:

```text
#A3EA2A
```

Structure:

```text
#RRGGBB
```

Where:

```text
RR = Red
GG = Green
BB = Blue
```

Each pair contains:

```text
2 hexadecimal digits
```

representing:

```text
1 byte
```

---

# Key Takeaways

- Humans primarily use the **Decimal (Base-10)** system.
- Computers operate using the **Binary (Base-2)** system.
- **Hexadecimal (Base-16)** provides a shorter and easier way to represent binary values.
- **Octal (Base-8)** groups binary digits in sets of three.
- A **Bit** is a single binary value (0 or 1).
- A **Byte** consists of 8 bits.
- Computer colors use the **RGB model**.
- A 24-bit RGB color can represent over **16 million colors**.
- Hexadecimal color codes are widely used in web development, networking, and cybersecurity.
