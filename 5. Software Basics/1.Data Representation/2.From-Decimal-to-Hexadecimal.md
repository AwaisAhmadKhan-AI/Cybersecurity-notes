# Numbers: From Decimal to Hexadecimal

## Overview

Humans naturally use the **Decimal System (Base-10)** because we have ten fingers.

Computers use the **Binary System (Base-2)** because electronic components only understand two states:

- ON / OFF
- High Voltage / Low Voltage
- True / False
- 1 / 0

Everything inside a computer is ultimately represented using **0s and 1s**.

---

# Decimal Numbers (Base-10)

A decimal number is made of powers of 10.

Example:

```text
213
```

Can be written as:

```text
213 = 2×10² + 1×10¹ + 3×10⁰
```

Calculation:

```text
= 2×100 + 1×10 + 3×1
= 200 + 10 + 3
= 213
```

---

# Binary Numbers (Base-2)

Binary only uses:

```text
0 and 1
```

Each position is a power of 2.

Example:

```text
1001
```

Calculation:

```text
1001
= 1×2³ + 0×2² + 0×2¹ + 1×2⁰
= 1×8 + 0×4 + 0×2 + 1×1
= 9
```

Therefore:

```text
1001₂ = 9₁₀
```

# Why Hexadecimal?

Binary numbers become very long.

Example:

```text
101000111110101000101010
```

Hard to read ❌

Hexadecimal provides a shorter representation.

# Binary to Hex Conversion

Group binary digits into blocks of 4 bits.

Example:

```text
101000111110101000101010
```

Split:

```text
1010 0011 1110 1010 0010 1010
```

Convert each group:

```text
1010 = A
0011 = 3
1110 = E
1010 = A
0010 = 2
1010 = A
```

Result:

```text
A3EA2A
```

---

# Hexadecimal to Decimal

Example:

```text
9BDF
```

Calculation:

```text
9×16³ + B×16² + D×16¹ + F×16⁰
```

Replace letters:

```text
B = 11
D = 13
F = 15
```

```text
= 9×4096 + 11×256 + 13×16 + 15×1
= 36864 + 2816 + 208 + 15
= 39903
```

Therefore:

```text
9BDF₁₆ = 39903₁₀
```

# Octal to Decimal Example

Example:

```text
357
```

Calculation:

```text
3×8² + 5×8¹ + 7×8⁰
```

```text
= 3×64 + 5×8 + 7×1
= 192 + 40 + 7
= 239
```

Therefore:

```text
357₈ = 239₁₀
```

# Key Takeaways

- Computers work using **Binary (0 and 1)**.
- Each binary position is a power of **2**.
- Hexadecimal groups **4 binary bits** into one digit.
- Octal groups **3 binary bits** into one digit.
- Hexadecimal is widely used in:
  - Colors
  - Memory addresses
  - Networking
  - Cybersecurity
- Understanding Binary and Hex is a fundamental cybersecurity skill.
