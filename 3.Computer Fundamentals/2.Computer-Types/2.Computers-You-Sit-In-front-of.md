# The Computers You Sit in Front Of

## Overview

During her first month at Nova Labs, Sophia learned that not all computers are the same.

---

## Key Learnings

1. Not all computers are meant to move  
2. Not all computers are meant for direct user interaction  

---

## Main Types of Computers

| Computer Type | Screen & Keyboard | Main Purpose |
|----------------|------------------|--------------|
| Laptop | Yes | Portable everyday computing |
| Desktop | Yes | Sustained performance at a fixed location |
| Workstation | Yes | Precision and reliability for professional tasks |
| Server | No | Providing services to many users over a network |

---

## Understanding Each Type

### Laptop
- Portable and lightweight
- Designed for everyday use
- Can slow down under heavy or long workloads
- Limited cooling due to compact size

---

### Desktop
- Fixed in one place
- Uses wall power (no battery limits)
- Better cooling system
- Performs consistently for long tasks

---

### Workstation
- Looks like a desktop but more powerful
- Built for professional workloads (e.g. 3D modeling, simulations)
- Focuses on **accuracy and reliability**
- Reduces errors in complex computations

---

### Server
- No screen or keyboard
- Runs continuously (24/7)
- Handles requests from many users at once
- Powers websites, apps, and online services

---

## Key Insight

Each computer type is designed for a **specific purpose**, balancing:
- Performance
- Portability
- Reliability
- Accessibility

---

## Key Takeaway

Computers are not one-size-fits-all — from laptops to servers, each type is optimized for a different role in the digital world.
