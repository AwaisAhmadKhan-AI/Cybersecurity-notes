## Why Computers Come in Different “Flavours”

Computers are designed differently because each one is optimised for a specific job, and improving one feature usually creates a trade-off elsewhere.

### Key Idea
Every computer design is a **trade-off** (gain in one area = loss in another).

### Main Trade-offs
- **Mobility vs Performance**  
  Smaller devices (phones, laptops) are portable and energy-efficient, but cannot sustain very heavy workloads for long periods.

- **Performance vs Cost/Reliability**  
  Servers and workstations are powerful and highly reliable, but they are expensive and not portable.

- **Purpose-driven Design**  
  Each type of computer is built for a specific role:
  - Phones → personal everyday use  
  - Servers → provide services to many users  
  - IoT devices → simple background tasks  

### Final Idea
There is no “best” computer.

There is only the **right tool for the right job**.
