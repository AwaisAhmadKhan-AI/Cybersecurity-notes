# Cloud Computing Overview

## What is Cloud Computing?

Cloud computing is a model where computing resources such as servers, storage, databases, and applications are delivered over the internet instead of being hosted on a local machine.

Instead of running everything on a single physical computer, applications use remote infrastructure that is accessible from anywhere.

---

## Evolution of Servers to Cloud

Cloud computing did not appear suddenly. It evolved over time as organizations tried to solve key problems:

- High cost of physical servers  
- Low hardware utilization  
- Slow deployment of systems  
- Difficulty in scaling applications  

This evolution moved from:
- Physical servers → Virtualization → Cloud computing

---

## Key Characteristics of Cloud Computing

### Scalability
Resources can be increased or decreased based on demand.

### On-Demand Self-Service
Users can create or remove resources instantly without manual setup.

### Pay-as-You-Go
You only pay for what you actually use.

### Security
Cloud providers implement strong infrastructure-level security.

### High Availability
Systems are designed to stay online even if parts fail.

### Global Access
Services can be accessed from anywhere in the world.

---

## Cloud Deployment Models

### Public Cloud
- Shared infrastructure managed by providers  
- Used by startups and global applications  
- Highly scalable and cost-effective  

### Private Cloud
- Dedicated infrastructure for one organization  
- Used in banking, healthcare, government  
- Offers higher control and security  

### Hybrid Cloud
- Combination of public and private cloud  
- Used when sensitive data must stay private while scaling globally  

---

## Cloud Service Models

### Infrastructure as a Service (IaaS)
- Provides virtual machines, storage, and networking  
- You manage: OS and applications  
- Provider manages: physical hardware  

---

### Platform as a Service (PaaS)
- Provider manages infrastructure + OS  
- You focus only on application development  

---

### Software as a Service (SaaS)
- Fully managed applications over the internet  
- Example: email services, video conferencing tools  
- You only use the software

## Key Idea

Cloud computing enables applications to be:
- Scalable  
- Cost-efficient  
- Globally accessible  
- Highly available  

It allows companies to focus on building products instead of managing physical infrastructure.
