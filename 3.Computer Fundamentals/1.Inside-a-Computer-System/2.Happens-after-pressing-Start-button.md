# What Happens When You Press the Power Button?

## Overview

When you press the power button, the computer goes through a **boot sequence** to start the operating system and prepare the system for use.

---

## Step 1: Power Button Pressed

- You press the power button
- Signal is sent to the **PSU (Power Supply Unit)**
- PSU starts supplying electricity to all components

👉 Like waking up and your body starts breathing again

---

## Step 2: Firmware Starts (UEFI/BIOS)

- **UEFI (modern)** or **BIOS (older)** starts running
- It acts as the system’s startup manager
- Initializes basic hardware

---

## Step 3: Power-On Self Test (POST)

- System checks if hardware is working properly:
  - RAM
  - CPU
  - Storage
  - Keyboard, etc.

👉 If something is wrong → system shows error/beeps

---

## Step 4: Boot Device Selection

- UEFI looks for a **boot device**
- Boot order decides priority:
  - SSD
  - HDD
  - USB

👉 It finds where the operating system is stored

---

## Step 5: Bootloader Starts

- Bootloader is loaded from the selected device
- It loads the **Operating System into RAM**
- Control is handed over to the OS

---

## Final Result

- Operating System starts (Windows, Linux, etc.)
- You see login screen or desktop

---

## Simple Flow

```
Power Button
→ PSU powers system
→ UEFI/BIOS starts
→ POST checks hardware
→ Boot device selected
→ Bootloader loads OS
→ Computer is ready
```

---

## Key Takeaway

Pressing the power button starts a **careful step-by-step process** that checks hardware, finds the operating system, and loads it into memory so the computer can run.
