# Inside a Computer System

## Overview

Every computer system is made of **core hardware components**, and each one has a specific job. Together, they allow the computer to function like a complete system.

---

## CPU (Central Processing Unit)

### What it does:
- Acts as the **brain of the computer**
- Performs calculations and executes instructions

👉 Without CPU: nothing can run

---

## RAM (Random Access Memory)

### What it does:
- Temporary short-term memory
- Stores data currently being used

👉 Faster access, but **data is lost when power is off**

---

## Storage (HDD / SSD)

### What it does:
- Long-term memory
- Stores files, apps, and the operating system

### Types:
- HDD → slower, mechanical
- SSD → faster, modern

---

## Motherboard

### What it does:
- Main circuit board
- Connects all components together
- Allows communication between parts

👉 Think of it as the **central highway**

---

## GPU (Graphics Processing Unit)

### What it does:
- Handles graphics and visual processing
- Used for gaming, videos, and rendering

---

## Power Supply Unit (PSU)

### What it does:
- Supplies power to all components
- Converts electricity into usable form

---

## Network Interface Card (NIC)

### What it does:
- Enables internet/network connection
- Provides MAC address for the device

---

## Cooling System

### What it does:
- Prevents overheating
- Uses fans or liquid cooling

---

## Simple Analogy (Human Body)

| Computer Part | Human Analogy |
|--------------|--------------|
| CPU | Brain |
| RAM | Short-term memory |
| Storage | Long-term memory |
| Motherboard | Nervous system |
| GPU | Eyes (visual processing) |
| PSU | Heart (power supply) |

---

## Key Takeaway

A computer works because all components **specialize in different tasks and work together like organs in a human body.**
